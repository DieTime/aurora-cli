# Install Aurora CLI

### Install dependencies

```shell
sudo apt update && \
  sudo apt install python3-pip git git-lfs curl tar unzip bzip2 ffmpeg
```

!!! info

    The application uses `sudo`, if you don't have it installed by default, install it.
