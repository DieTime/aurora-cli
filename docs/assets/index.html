Open-source software
An application that provides an easy start in the Aurora OS ecosystem. Work with Aurora SDK, Aurora Platform SDK, Flutter SDK, interaction with devices, emulator and much more.

Easy installation of the necessary tools of the Aurora OS ecosystem.
Installation in one terminal command: Aurora SDK and emulator, Aurora Platform SDK, Flutter SDK with support for the Aurora OS platform. Management of installed SDKs, expanding their functionality and simplifying work with them.
The application is in Python and is available on PyPi.
Install
Available application for most Linux distributions.
Python 3.8 support allows you to work with the application on most modern Linux distributions, such as Ubuntu 20.04+, ALT Linux 10+, and others. Installation of the application is available via Python package-management pip.
