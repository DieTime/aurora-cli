Добавьте устройства с ОС Aurora во Flutter.

#### Options

```shell
-s, --select         Выберите из доступных.
-i, --index INTEGER  Укажите индекс.
-v, --verbose        Подробный вывод.
--help               Показать это сообщение и выйти.
```

#### Example

```shell
aurora-cli flutter custom-devices
```
