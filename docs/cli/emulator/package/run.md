Запустите пакет на эмуляторе.

#### Options

```shell
-p, --package TEXT     Имя пакета.  [обязательно]
-m, --mode [dart|gdb]  Режим запуска приложения.
-v, --verbose          Подробный вывод.
--help                 Показать это сообщение и выйти.
```

#### Example

```shell
aurora-cli emulator package run
```
